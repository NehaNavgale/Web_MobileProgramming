This page has been created to document the learnings of In class programming 1. The lecture 1 included the introduction to Github and installation of Webstorm and Github. Following are the points we covered in lecture 1.

  • Introduction to GitHub
  • Installation of Webstorm and GitHub Desktop
  • Creation of Github repository
  • Sync local and remote repository
  • Creation of index.html to document lecture 1
  • Fork existing repository
  • Create a branch for your repo and make a pull request
  • Create an issue and describe your pull request

Following are the steps followed to create repository

  • Create new repository in GitHub
  • Clone the repository in local and open it in Desktop
  • Create Source and Documentation folder on local
  • Add index.html file in source folder and add all the screenshots in documentation folder. Commit the changes to remote repo
  • Fork existing repository.
